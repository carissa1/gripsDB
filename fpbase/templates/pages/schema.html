FPbase Database Schema

FPbase is a database designed specifically for fluorescent proteins. The goal is to come up with a single database design that can categorize the majority of the many subtle properties that fluorescent proteins can possess. This site is designed by microscopists, so currently the emphasis is on properties that affect the usage of fluorescent proteins. If you have suggestions for ways to extend the database model to incorporate additional properties (e.g. protein structure, etc...), feel free to contact us.
